Important People

HTML - If you want to login to a website, you send a get request and the server responds with HTML for the login page. HTML stands for HyperText Markup Language which essentially is the language used to tell a web browser how the page should look, used in fonts, links, typographical emphasis, images, alignment, etc. The browser sends HTTP requests to the server and the server responds in HTML.

WC3 - is a community of international standards with a full time staff directly working with the public with the goal to develop Web standards established to lead the web to its truest potential.

CSS - Cascading Style Sheets are used to build design websites similar to HTML. While HTML is the arrangement of web documentation, CSS gives specifications to the style of the documents including layouts, colors, fonts, etc. HTML provides the structure of the page, CSS the (visual and aural) layout, for a variety of devices.
