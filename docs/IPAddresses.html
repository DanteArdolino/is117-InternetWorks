IP Addresses, Packets and Routing

The internet is a design philosophy and architecture expressed in a set of protocols. A protocol is a well-known set of standards that if all parties agree to use it will allow them to communicate without trouble. All devices connected to the internet have unique addresses which are a series of numbers. The addressing system for devices on the internet forms the most important protocols called the Internet Protocol or IP.

A packet is a small amount of data sent over a given network. Every packet includes a source and destination as well as the data that will be transmitted.The packets carry the data in the protocols that the Internet uses: Transmission Control Protocol/Internet Protocol. Each packet is then sent off to its destination by the best available route. Routers are used to determine such, as they examine the destination IP address and make their routing decisions accordingly; this is done by use of routing tables which list all networks for which routes are known.
