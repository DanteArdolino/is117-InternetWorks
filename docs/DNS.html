DNS

Domain Name System is one of the most important and possibly the most overlooked aspects that involve the internet. Without DNS, the internet would crumble. Computers are set up in large networks that communicate with each other via wires identified using IP addresses. DNS is used to translate an actual name into these numbers (IP addresses). When typing in an address such as www.google.com , there is actually a hidden “.” at the end which is the root of the internet's namespace. Operating systems are configured to ask a resolving name server for IP addresses it does not know which is part of the DNS. Resolving name servers only know how to find the root name servers.

This is important and we use DNS because it is a central part of the internet. It provides a way to match names of the websites to the numbers of the website. Doing so enables humans to use easily recognized and memorized domain names while the computers on the internet can use numbers like IP addresses.
