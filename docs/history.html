History of the Internet

  • Oct 4, 1957
    Event One

    Russia puts Sputnik into orbit.

  • Feb 7, 1958
    Event Two

    Space exploration assigned to Advanced Research Projects Agency (ARPA)

  • Oct 1, 1958
    Event Three

    NASA opens for business via Eisenhower executive order

  • July 1961
    Event Four

    Leonard Kleinrock of MIT – First paper on packet switching theory.

  • August 1962
    Event Five

    J.C.R. Licklider of MIT – memos on “Galactic Network Concept”

  • Oct 1969
    Event Six

    Larry Roberts – heads project that connects first two ARPA NET sites – UCLA and Stanford Research Institute.

  • Oct 1972
    Event Seven

    Bob Kahn – demonstrates ARPANET at the International Computer Communication Conference (ICCC)

  • March 1972
    Event Eight

    Ray Tomlinson – First basic email program using “@”

  • May 1974
    Event Nine

    Vinton Cerf & Ray Kahn - A Protocol for Packet Network Interconnection

  • 1978
    Event Ten

    TCP and IP split

  • 1980
    Event Eleven

    Tim Berners-Lee – writes predecessor to WWW

  • Nov 1983
    Event Twelve

    Jon Postel, Paul Mockapedis, and Craig Partridge design the Domain Name System (DNS)

  • 1990
    Event Thirteen

    Tim Berners-Lee creates World Wide Web

  • 1993
    Event Fourteen

    Marc Andreesen – Develops Mosaic web browser

  • 1996
    Event Fifteen

    Microsoft vs Netscape - Browser wars
